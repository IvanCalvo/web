Trabalho 1 - Desenvolvimento de Software Web 1 

Website for Car Dealing - CarDealership

User roles:
Admin
Cliente
Loja

Loja can register cars
Cliente can make one proposal per car
Loja than can Accept or refuse that proposal, if refusing can optionaly make a counter proposal.
Cliente can see a list of all cars by all Lojas and filter by model.

Users not logged can see the list of cars and filter by model, but can't create proposals.
